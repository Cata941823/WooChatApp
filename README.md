# WooChatApp
WooChatApp este un proiect pentru facultate, ce contine JAVA si SQL.

Cerinte principale:
 - login ........................................................... 100%
 - register ........................................................ 100%
 - cautare useri ................................................... 100%
 - conversatie intre doua sau mai multe persoane.................... 70%
 - trimis poze ..................................................... 80%
 - schimbarea titlului conversatiilor .............................. 60%
